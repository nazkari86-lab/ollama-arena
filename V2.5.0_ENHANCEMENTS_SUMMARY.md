## 1. Genome Lineage Auto-seed ✅

### Implementation

**New File: `ollama_arena/genome/lineage_inference.py`**
- Created comprehensive lineage inference engine with 328 lines of code
- Implemented multiple heuristic algorithms for detecting model relationships:
  - **Pattern Matching**: Analyzes naming conventions (e.g., `-instruct`, `-distill`, `-coder`)
  - **Architecture Similarity**: Compares layers, hidden size, context length, vocab size
  - **Parameter Relationships**: Detects distillation (size differences) and fine-tuning (similar sizes)
  - **Metadata Clustering**: Groups by family and organization
- Confidence scoring system (0.0-1.0) for each hypothesis
- Supports dry-run mode for previewing hypotheses before applying
- Relation hierarchy: `fine_tuned_from`, `distilled_from`, `merged_from`, `quantized_from`, `trained_from`

**Key Classes:**
- `LineageHypothesis`: Dataclass for proposed lineage relationships
- `LineageInferenceEngine`: Main inference engine with heuristic algorithms

### Usage Examples

```bash
# Automatic lineage inference
ollama-arena genome auto-seed --min-confidence 0.5

# Preview without applying
ollama-arena genome auto-seed --dry-run

# Custom confidence threshold
ollama-arena genome auto-seed --min-confidence 0.7
```

### Algorithm Details

The inference engine uses a multi-factor approach:

1. **Naming Pattern Analysis** (0.3-0.4 confidence)
   - Suffix patterns: `-instruct`, `-chat`, `-distill`, `-coder`, `-turbo`
   - Prefix patterns: family name matching
   - Normalization removes quantization tags and version numbers

2. **Architecture Similarity** (0.2-0.7 confidence)
   - Hidden size similarity (<10% difference)
   - Context length comparisons (child often > parent)
   - Layer count analysis
   - Vocab size matching (should be identical or similar)
   - MoE structure detection

3. **Parameter Relationships** (0.3-0.6 confidence)
   - Distillation: child size < parent size * 0.5
   - Fine-tuning: similar size (<10% difference)
   - Merged models: MoE structure detection

4. **Metadata Clustering** (0.1-0.3 confidence)
   - Same family grouping
   - Same organization grouping

Confidence scores are combined and only hypotheses above threshold are applied.
